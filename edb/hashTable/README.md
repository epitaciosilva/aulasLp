## Hash table

Hash table ou tabela de espalhamento é uma estrutura de dados que armazena elementos de forma espalhada, mas não aleatoriamente.
A ideia é separar os elementos de acordo com uma certa características chave deles.
Assim, eles são separados em certas categorias.
Dessa forma, a busca só é feita dentro da categoria em que o elemento está inserido, diminuindo a complexidade de busca em comparação a uma lista.
