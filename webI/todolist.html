Lista de x tarefas a fazer usando HTML

Desenvolver, utilizando apenas tecnologias Web lado-cliente, uma lista de tarefas-a-fazer (To-Do List) que permita editar (inserir, remover e alterar) tarefas, incluindo a descrição da tarefa, a quem ela deve ser atribuída e o prazo previsto para a sua realização.
Na primeira destas atividades, você vai usar APENAS HTML. Nas aulas seguintes, vamos usar JS para permitir a interatividade.
